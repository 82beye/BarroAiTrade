# 주식단테 수박지표 기반 당일 자동매매 시스템

## 시스템 개요
주식단테의 '역매공파' 이론에서 파생된 **파란점선**과 **수박지표**를 활용하여,
당일 산출된 종목을 자동 매수하고 **15:00 장 마감 전 전량 청산**하는 데이트레이딩 시스템.

## 핵심 원칙
- **데이트레이딩 전용**: 당일 매수 → 당일 청산 (오버나이트 없음)
- **14:50 강제 청산**: 어떤 상황에서도 14:50에 보유 포지션 전량 시장가 매도
- **키움 REST API**: openapi.kiwoom.com 기반 (Python, 크로스플랫폼)
- **모의투자 우선**: 반드시 모의투자로 충분히 검증 후 실거래 전환

## 매매 전략 요약

### 종목 산출 (08:30 Pre-Market)
1. KOSPI/KOSDAQ 전종목 일봉 데이터로 지표 계산
2. **파란점선 돌파 임박** 종목 필터링 (전일 종가가 파란점선의 98~102% 범위)
3. **수박지표 발생** 종목 우선순위 부여
4. 시장 리스크 필터 적용 (코스닥 20일선 체크)
5. 최종 감시 리스트 10~20종목 산출

### 매수 조건 (09:00~14:30)
- 파란점선 상향 돌파 확인 (현재가 > 파란점선)
- 거래량 폭증 확인 (당일 거래량 > 20일 평균 × 3.0)
- 양봉 확인 (현재가 > 시가)
- 종목당 총 자산의 10% 이내, 최대 5종목

### 매도 조건
- **익절**: +3% 도달 시 50% 매도, +5% 도달 시 나머지 전량 매도
- **손절**: -2% 도달 시 전량 매도
- **강제 청산**: 14:50 시장가 전량 매도 + 미체결 전량 취소

```
ai-trade/

├── config/
│   ├── settings.yaml         # 매매 파라미터 설정
│   └── .env.example          # API 키 템플릿
├── scanner/
│   ├── indicators.py         # 파란점선 + 수박지표 계산 엔진

├── strategy/
│   ├── entry_signal.py       # 매수 신호 생성
│   └── exit_signal.py        # 매도/청산 신호 생성
├── execution/
│   ├── kiwoom_api.py         # 키움 REST API 클라이언트
│   ├── order_manager.py      # 주문 생성/수정/취소
│   └── position_tracker.py   # 실시간 포지션 추적
├── monitoring/
│   ├── telegram_bot.py       # 텔레그램 알림
│   └── daily_report.py       # 일일 수익 리포트
├── main.py                   # 메인 실행 파일
└── docs/
    └── INDICATOR_SPEC.md     # 지표 수식 상세 문서
```

## 실행 방법
```bash
# 1. 환경 설정
cp config/.env.example config/.env
# .env 파일에 API 키 입력

# 2. 의존성 설치
pip install -r requirements.txt

# 3. 모의투자 모드로 실행
python main.py --mode simulation

# 4. 실거래 모드 (충분한 검증 후)
python main.py --mode live
```

## 에이전트 행동 규칙
1. **절대로** 14:50 이후까지 포지션을 보유하지 않는다
2. **절대로** 종목당 총 자산의 10%를 초과하여 매수하지 않는다
3. **절대로** 동시에 5종목 이상 보유하지 않는다
4. 모든 주문은 로그에 기록한다
5. 텔레그램으로 매수/매도/청산 알림을 발송한다
6. 오류 발생 시 즉시 전 포지션 청산 후 시스템 중단
