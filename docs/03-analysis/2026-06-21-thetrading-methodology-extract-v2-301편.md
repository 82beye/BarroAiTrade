## 0. 분석 방법 및 한계

- 5개 시기 배치(2021-10~22-09 / 22-09~24-03 / 24-03~25-05 / 25-05~26-04 / 26-04~26-06)를 병렬 채굴, R1~R13 대조 후 통합.
- **한계**: 자동자막 노이즈(종목명·수치 오인식), 화자가 다수(기법 혼재), 채널 엣지의 상당부분이 재량적(실시간 뉴스·테마 판단)이라 정량 전이는 (A)분류 한정. 수치는 화자 발화 인용 기반 추정 구간.
- **검증 일관성**: 채널 용어가 BarroAiTrade 코드 전략명과 1:1 대응(F존=`f_zone`, 골드존=`gold_zone`, SF존=`sf_zone`, 종가베팅=`closing_bet`)함은 선행 문서에서 확인됨. 본 델타는 그 위에 임계·신규규칙을 얹는다.

## 1. R1~R13 정제 (풀코퍼스 재검증 + 갱신 임계)

99편 추출의 13개 규칙은 **전 시기에서 재확인**됐다. 풀코퍼스로 드러난 **구체 임계 갱신**:

| # | 규칙 | 99편 정의 | **301편 정제·갱신 (신규 수치 굵게)** |
|---|------|-----------|----------------------------|
| R1 | F존 얕은눌림 반등 | 매수 | 1차 패스→**2차 F존 진입** 전술 정착. 분할(선진입 1/3→도달 시 추가). 티마앱 정의 **"B1 기준 ±2% 이내, -2% 추가이탈 or 일봉 20일선 이탈 시 퇴출"** |
| R2 | 골드존 0.5~0.618 | 코어존 | **15분봉 기준봉 고점→저점 피보, 0.5·0.618 두 레벨만**. 1차=0.5, 2차=0.618. SF존∩골드존 겹침=최우선. 단타 성공률 **72%**(티마 통계 24.10~25.08) |
| R3 | 0.618 이탈 손절 | 손절 | "최후 보루 0.618" 재확인 + **일봉 20일선 이탈 동급 손절선**. 실전 조기손절 **-3~4%** 병행 |
| R4 | 분봉 자금유입 | 오전1+오후1 | **9~10시=매도구간(신규 진입 금지), 10~14:30=관찰만, 14:30~15:20=매수**. "오전유입 후 오후 死 = 종베 금지", "오전+오후지지=A급" |
| R5 | 대장주 거래대금순위 | 1~5위 강제 | **"5위 밖 = 주도주 아님, 무조건"**(가장 강한 정의). 종베 시점(3시)에도 5위 재확인 |
| R6 | 신고가 필수+가점 | 비중 가점 | **신고가 판정 = 꼬리 제외 몸통(종가) 기준**. **3요소 동시충족(거래대금 최다 ∧ 몸통 신고가 ∧ 현재 이슈) → 1개 미달 즉시 제외**. 정배열+**52주 신고가**(버저비터) |
| R7 | 거래대금 절대액 | ≥300억 | **2026 실질 기준 조 단위**(후보군 ≥1조). **1분봉 ≥15억**. 시총 하한 **≥1,000억**(선수 미진입), 2026 대형주 ≥10조. 코스닥 총대금 온도계: 5조 약세/7~8조 적정/10조+ 풍부 |
| R8 | 호가밀집 A급 | 호가 두께 | **1억 클릭 시 호가당 ≥762주(~10억/호가)**. 상한가 잔량 A급 **≥130만주(~100억)**. **잔량 10% 소진 시 자동매도**(HTS 0317). "돼지물량"=두꺼운 매수벽→반등 타점 |
| R9 | 종가베팅 | 15:00~15:20→익일슈팅 | **KRX 15:00~15:20 + NXT 19:30~19:50 이원화**(본장 1/3 + NXT 2/3). 익일 **NXT 8:00 매도** 루틴. 목표 평균 **+4%**. 동시호가 15:20 |
| R10 | 보유 한도 | D1~D3, 4일차 5일선 | **종베는 3일차까지만**(4일차=5일선 붙어 무너지는 습성). **3일선 반등**(신고가 장대양봉 익일) |
| R11 | 진입가 위치 | 불나방 역매수=고점추격 금지 | **잔존 기대수익 게이트**(먹을 구간 50%↓ 소진 시 NO). "현재가 위로 따라사면 사자마자 손실". 2·3등주 금지 |
| R12 | 종목압축 | 1~2개 | **원픽(1종목)** 또는 당일주도주+전일주도주 2개. **4종목↑=집중력 희석/주문 실기**. 계좌 분리(종베 전용+관찰) |
| R13 | 연속손실 STOP | 3회 STOP·수익보존 | 뉴하트 "2일 연속손실→3일째 무조건 +". 버저비터 컷 **-300~400만**. **연속수익→자만→비중확대→대형손실** 고리. 수익 즉시 인출(생명줄) |

## 2. 신규 규칙 R14+ — 정량·코드화 후보 (델타)

|----|------|---------------|-----------|
| **D-R14** | 잔존 기대수익 게이트 | `remaining_upside < entry_upside × 0.5` → NO-GO ("30% 중 15%만 남으면 진입X") | 2021~22 |
| **D-R15** | 시간 손절(Time Stop) | 진입 후 일정 시간 내 방향 미발생 → 청산 | 2021~22 |
| **D-R16** | 프로그램 상한가 익일 패대기 | 전일 프로그램 대량매수 상한가/장대양봉 → 익일 추격 금지(매도경향) | 전 시기 |
| **D-R17** | 2차 VI 직전 반등 | 1차 VI 통과 후 2차 VI 가격 직전 반등 경향 → 2차 직전 진입 | 전 시기 |
| **D-R18** | 라운드피겨 지지/저항 | 정수 가격대(00 끝)에 물량 집중→지지·저항 타점 | 전 시기 |

## 4. 2025~2026 구조 변화 (방법론 진화)

1. **NXT 대체거래소 도입** — 종베 시간 이원화(KRX 15:00~15:20 + NXT 19:30~19:50), 본장 1/3 + NXT 2/3, 익일 NXT 8:00 매도, 금요일 NXT 최저 16:40~16:50. 상따 갭 약화로 최고점 상따 비중 축소. → **`closing_bet` 진입/청산 시간창 파라미터에 직접 영향**.
2. **거래대금·시총 기준 대폭 상향** — 조 단위 정상화, 시총 10조+ 선호. "테마주·세력주 쇠퇴, 대형주·실적주 중심(나스닥화)". → **R7/선정 임계 재설정 필요**.
3. **로봇/피지컬AI 신규 주도** — 반도체와 병립. 섹터 순환(정치→조선/방산/원전→로봇/AI→ADC바이오/우주/양자→반도체/로봇 재주도).
4. **티마(TIMA) 앱 도구화** — F존 인박(B1±2%), 골드존/38스윙 성공률 통계(72%/78%), 별표 테마, 2년내 상한가 횟수(끼 판별). 채널이 자체 룰을 앱 파라미터로 정량화함 → **코드화 가능성 시사**.
5. **시황→섹터→종목 3단 커리큘럼 공식화**("메스" 시리즈 2026.06~).

## 8. 부록 — 영상 단위 정밀 패스 (b03 종가베팅 강의군 정밀 채굴)

배치 단위 채굴(§1~7) 위에, 종가베팅 강의 영상(2025~2026, 뉴아트/더트레이딩)을 **영상 단위로 정밀 재분석**해 얻은 추가 수치·신규 규칙. 자막 실발화 근거.

### 8.1 R1~R13 정밀 정제
- **R2/R3 — 0.5 "주포 관리선"**: 종배 진입 가부 = **15분봉 피보나치 0.5(양봉 중심값) 미이탈**. "0.5 하향이탈 = 주포 이탈 = 주도 종배 아님"(0.618 손절보다 보수적 선행 게이트). 테마·거래대금 무관 0.5 우선.
- **R6 — 신고가 룩백 121일(약 6개월)**, 판정 "검은선(전고 라인) 돌파"(시각). 신고가 장대양봉 → 익일 **3일선** 반등.
- **R8 — 호가 1억클릭 A+/A/B 등급법**: 1억에 해당하는 주식수 확인 → **위·아래 둘다 빡빡=A+, 한쪽만=A, 위아래 비면=B(선수 미진입)**. **B급은 종배 제외**. "돼지 호가창"(대량물량 라인)=매수 타점(퍼센트 말고 호가세). 1등 상한가 못 샀으면 같은 테마 **호가 A급 2등주를 익일 종배**.
- **R9 — 종배 2타점 분할**: 15:00(본장) + **19:50(NXT)**, 비율 1/3 + 2/3. 목표 **+4% 일괄청산**(손절도 ~4%). 우선순위: 당일 주도주 없으면 전일→전전일 롤백, 없으면 무포.
- **R10 — 3일선 우선(4일차 5일선보다)**: 일봉 3일선/5일선 = **15분봉 60선/120선**. 종배 주력 지지 변수는 3일선.
- **R13 — 10시 손절룰**: 종배 익일 **오전 10시**까지 방향 안 나오면 손절. "10종목 중 3건 손절은 정상"(자연 발생률). 손절 준수가 계좌 향방 결정.
