## 0. 개요 — 대상·표본·방법·주의

- **대상**: 한국 주식 유튜브 '주식단테'. 코퍼스 `/Users/beye/data/youtube_script/주식단테/` (공개자막 1,010편, 14.2M자, 2015-02~2026-06).
- **방법**: 기법 키워드 밀도 정찰 → 시그니처 기법별 최고밀도 영상(~30편) 8개 클러스터 병렬 채굴 → 정량 메커니즘만 추출(세력 음모론·동기부여 멘트 제외).
- **채널 성격 주의 (전이 시 필수 인지)** — [[2026-06-17-thetrading-methodology-extract|더트레이딩]]과 달리 주식단테 공개영상은 **유료 멤버십 퍼널의 미끼층**이라는 구조적 편향이 있다. 추출된 "기법"의 상당 부분은:
  1. **검증 불가 서사**: "세력이 개미 물량을 흡수한다" 등 시장조작 가설 → 백테스트 무관, 패턴만 재해석해 사용.
  2. **비공개 핵심 지표**: '수박지표', '파란 점선', 검색기 "가린 버전" → 공개 발화로 복원 불가한 항목 존재.
  3. **재량적 판정**: 밥그릇 1/2/3/4번 자리 구분, "굽 높이", "안착" 등 후행(after-the-fact) 레이블 → 실시간 자동화 난이도 높음.
  4. **규제 리스크 화법**: 단정·확언("무조건/100%")·실시간 종목추천 → 채널 리스크일 뿐 코드 전이 대상 아님.
- 따라서 본 추출은 **정량화 가능 부분만 전이**하고, 각 규칙에 백테스트 가능성·비판메모를 병기한다.

## 1. 두 채널의 관계 (분석의 출발점)

[[../02-design/features/2026-06-17-thetrading-methodology-uplift.design|기존 설계 §0.1]]에서 확인된 사실: **BarroAiTrade의 전략명·로직은 '더트레이딩' 채널 방법론의 코드화**다(F존/골드존/SF존/supertrend = 더트레이딩 용어 1:1). 즉 이 시스템엔 이미 한 채널의 방법론이 깊게 이식돼 있다.

→ 따라서 주식단테 분석의 핵심 질문은 **"이미 더트레이딩이 이식된 시스템에 주식단테가 *무엇을 새로* 추가하는가"**이다. 단순 기법 나열이 아니라 **델타(신규/중복) 식별**이 목적이다.

## 2. 주식단테 시그니처 스택 (한눈에)

```
            ┌─────────────────────────────────────────────────┐
   장기축    │  EMA 112 / 224(세력선) / 448  ─ 추세 레짐·위계 지지 │  ← BarroAiTrade 부재(신규)
            └─────────────────────────────────────────────────┘
                       │ (역배열 바닥에서)
   패턴축    공구리(S-R flip) → 매집봉(흡수) → 밥그릇(원형바닥 4단계) → 하이힐(낙폭과대 반등 수렴돌파)
                       │
   통합셋업  역매공파 = 역배열 + 매집봉 + 공구리 + 볼린저 수축→팽창 + EMA112 근접   (손절3.5%/목표20%)
                       │
   스캐너    오돌이 시초가 단타 검색기(10조건) · 기준봉 224돌파 검색기
                       │
   운영      수급게이트(신용잔고·ETF·쏠림) · R:R≥1:2 · 분할매수 · 칼손절 · 몰빵금지 · 고점추격금지
```

**핵심 대비**: 더트레이딩/현 시스템 = *상승추세 종목의 돌파후 눌림(연속형)·단기선(5·20·60)*. 주식단테 = *역배열 바닥종목의 장기선(112/224/448) 재돌파(반전형)*. **방향이 반대인 보완적 셋업 클래스**.

## 3. 규칙 카탈로그 (JD-R) — 그룹별

> 표기: 정의(표준 TA) / 정량조건 / 손절·익절 / 핵심 파라미터 / 백테스트[가능·부분·불가] / BarroAiTrade 매핑 / 인용(videoID) / 비판.
> 자동자막 오인식 보정 후 **실제 발화 수치만** 수록.

### 그룹 A — 장기 이평 골격 (EMA 112 / 224 / 448)

**JD-R1 · 224세력선 추세 레짐 게이트** — *신규*
- 정의: 224일 EMA를 장기 추세선("세력선")으로. 위=지지/공격, 아래=저항/방어. 위계 112(1차)·224(2차)·448(3차).
- 정량: 224 위 유지=상승레짐 / 아래=하락레짐. EMA 사용 명시("지수로 고치세요").
- 백테스트: **부분** ("안착" 판정 기준 미정의).
- 매핑: 신규 추세 레짐 게이트(종목별). 기존 macro `market_regime.py`(지수 국면)와 **다른 차원**(종목 가격구조).
- 인용: KHiuCH2GEDw "224일선 밑에서 4개월 이상 있던 종목이 224일선 부근했을 때 관심을 가져라" / 지수EMA 권장.
- 비판: "세력 안착" 주관. 10% 반경 동적. 4개월 lookback 소형주 사이클 과최적화 위험.

**JD-R2 · 224 돌파→눌림 재진입** — *신규*
- 정의: 224 아래 4개월(≈80봉)+ 횡보 → 상향 돌파(돌파봉=뚫음봉) → 224 위 눌림에서 재진입.
- 손절: **돌파봉 시가 이탈**(% 미언급). 익절: 다음 위계선(224→448) / 50~100%.
- 백테스트: 부분.
- 매핑: 눌림목 진입 트리거의 기준선 레이어.
- 인용: 4kwgvXP5FdU "224 밑으로 강하게 하락…4개월 이상 횡보…다시 224 뚫어 올리며 흔들기(3번)…안착하면 슈팅(4번)".

**JD-R3 · 이평때리기 (역배열 mean-reversion)** — *신규*
- 정의: 역배열 448>224>112 확장 후 주가가 최하단 112 EMA에 수렴·터치 시 기술적 반등 롱. "데드기법"의 반대.
- 정량: 448>224>112 + 3선 이격 확장 후 수렴 + 종가 112 터치 + 양봉/공구리 지지.
- 손절: 112 종가 재이탈(암묵). 목표: 112→224(`공짜 라인`)→448.
- 백테스트: **가능**.
- 매핑: `blue_line` 확장(현재 5/20 EMA) 또는 신규 장기 mean-reversion.
- 인용: KcytXTtDkic / JQJMyTkl_uo "112일선 뚫는 세력…실패해도 224까지 결국 간다".

**JD-R4 · 이평선 3개 계단식 돌파** — *신규(JD-R3와 동일 골격)*
- 정의: 112/224/448 배열(정/역)로 추세판단 + 각 선을 계단식 지지·저항 목표로. "첫 번째 밀리고 두 번째 뚫고".
- 백테스트: 가능. 매핑: EMA(8/21/55) 대신 (112/224/448) 파라미터 테스트.

### 그룹 B — 바닥축적 가격구조 패턴

**JD-R5 · 밥그릇 (원형바닥 4단계 메타패턴)** — *신규*
- 정의: 원형바닥(Saucer)+컵앤핸들 변형. 1번 급락 → 2번 저점 매집 횡보(224 아래) → 3번 224 돌파+눌림(진입) → 4번 슈팅.
- 정량: 224 아래 4개월+ 횡보; 돌파 강도 = 직전 이격 **2배 이상**; 코스닥 세력주는 2번 길이>1번(삼각형 면적). 목표 50~100%.
- 손절: 3번 돌파 직전 저점(공구리 라인) 이탈.
- 백테스트: **부분** (4단계 자동 레이블이 후행적, 난이도 중~고).
- 매핑: 신규 `saucer_bottom` 디텍터.
- 인용: fshUGuAWrho "1번보다 2번이 길어야…3번 자리는 6개월 들고 50% 먹는 자리" / IZ_qmX_sKII "이격보다 2배 이상 뚫어야".
- 비판: 작전 가정 검증불가(수급불균형 회복으로 재해석). 사후선별 편향(급등주에서만 패턴 보임).

**JD-R6 · 매집봉 (흡수 캔들)** — *신규*
- 정의: 전고 매물대까지 급등 후 되돌리는 캔들군(Volume Climax + 긴 위꼬리 페이크아웃/흡수). 224 아래·밥그릇 2번에서 발생.
- 정량(형태 3종): A 장대양봉형(≈25~30% 급등 후 시가 근처 되돌림, 긴 위꼬리) / B 연속양봉2+연속음봉2 / C 갭상승 후 밀어내림. 무뉴스 급등 선호. 출현 1~2회.
- 백테스트: **부분** (매집봉 거래량 배수 미명시; 위꼬리 되돌림율은 코딩 가능).
- 매핑: 신규 `accumulation_candle`(고가대비 종가 되돌림율 + 거래량 스파이크).
- 인용: IZ_qmX_sKII "개미 물량을 흡수하는 캔들…25%나 상한가 근처 갔다가 쭉 빠짐".

**JD-R7 · 공구리 (S-R 역전 / Support-Resistance Flip)** — *신규, 가장 정량적*
- 정의: 하락 파동 중 직전 전고점(언덕)을 상향 돌파 → 그 가격이 지지로 역전. "기초공사".
- 정량: 하락파동 내 Swing High 상향 돌파; 돌파 폭 > 직전 하락 폭; 진입=**돌파봉 시가**(또는 flip선 눌림 재진입); 실패=돌파 후 즉시 재이탈.
- 손절: 공구리 라인(구 전고점) 하향 이탈. 목표: 직전 하락폭 **대칭**. 유효기간 1~2주(단타·스윙).
- 백테스트: **가능** (Pivot/zigzag + 가격비교만으로 구현; 분봉~일봉 유연).
- 매핑: 신규 `sr_flip` 디텍터. 기존 zone(지지-저항) 전략에 직접 이식.
- 인용: 3zWm0BoSnMI "뚫어올린 봉의 시가를 던질 라인 잡고 매매" / yW3vrHijOBY "직전 언덕 뚫어올리면 공구리=지지라인".

**JD-R8 · 하이힐 (낙폭과대 반등 수렴 돌파)** — *신규*
- 정의: 돌발 악재 급락 → V/U 반등 → 수렴 박스(굽) → 박스 상단(발끝) 돌파. 공구리의 업그레이드.
- 정량: 낙폭과대(N일 -X%) → 반등(M일 +Y%) → 삼각수렴(LH↓+HL↑) → 상단 돌파봉 진입. "굽 높을수록 신뢰도".
- 손절: 수렴 박스 하단/굽 저점. 백테스트: **부분** (돌발악재·굽높이 정성).
- 매핑: 신규 `high_heel`(낙폭과대 반등 + 수렴 돌파).

### 그룹 C — 통합 시그니처 셋업

**JD-R9 · 역매공파(영매공파)** — *신규, 채널 플래그십 스윙*
- 정의: **역**배열 + **매**집봉 + **공**구리 + **파**(볼린저 수축→팽창 전환) + **EMA112 근접** 동시충족 스윙 롱.
- 정량: 448>224>112 역배열 + 매집봉 + 공구리 박스(하단=손절) + 볼린저 squeeze 해소 + 112 터치.
- 손절: **~3.5%**(공구리 박스 하단) / 목표: **~20%**(112→224) = **R:R ≈ 1:6** (유일하게 명시 R:R).
- 백테스트: **부분** (매집봉 정량화가 취약점 → 거래량 Z-score/OBV 대리). 매핑: `swing_38` 확장 또는 신규 모듈.
- 인용: fqaOvEhGrvQ "역·매·공·파 112…손절 3.5% 성공하면 20%".

### 그룹 D — 검색기(스캐너) 조건식

**JD-R10 · 오돌이 시초가 단타 검색기 (10조건)** — *신규, 최고 이식가치 단일 산출물*
- 대상: 장기 역배열로 충분히 빠진 종목이 5일선을 뚫어 올리는 바닥반등주(시초가 단타).
- 조건: ① 112/224/448 역배열 ② 갭 2~10%(≤5% 이상적) ③ 장초 1~2분 거래량 전일比 ≥50%(종가까지 ≈300% 예측) ④ 전일 종가 < 5일선 ⑤ 당일 종가 ≥ 5일선(돌파) ⑥ 체결강도 ≥ 110 ⑦ 매도호가잔량 ≥ 매수호가잔량 ×150% ⑧ 시가대비 ≥ 4%(주도주판 7~8%) ⑨ 5일평균 거래량 ≥ 5만주 ⑩ 5일평균 거래대금 ≥ 5억 + 제외(관리·투자위험·정리·ETN/ETF·환기·투자주의·초저유동성·권리락).
- 백테스트/스캐너: **가능** (①~⑤,⑧~⑩ OHLCV/이평으로 완전; ⑥체결강도=키움전용→근사, ⑦호가잔량=L2 실시간 필요).
- 매핑: `signal_scanner` long_condition_stack(AND) + 단타모드 게이트. 손절=역전봉 저가.
- 인용: zi0ouegiQcc. 비판: "가린 버전"으로 일부 세부 비공개; 뉴스필터 미포함.

**JD-R11 · 기준봉 224돌파 검색기** — *부분(핵심수식 2개 비공개)*
- 조건: 10봉내 종가 전일比 +10% & 거래량 전일比 200% 봉(기준봉) + 종가/224 ≤ 108~109%(근접) + 종가 > 224 + 5일평균 거래량 ≥ 7만주 + 112/224/448 OR. 힌트: "112/224/448 골든크로스 + 거래량 300% 돌파".
- 백테스트: 가능(부분 제약). 인용: CWF9OfB59WM.

### 그룹 E — 종목 성격별 진입 차별

**JD-R12 · 양봉/음봉 차별 (시총 분기)** — *부분*
- 정의: 시총 ≥ 1조 우량주 = **음봉 매수**(장대양봉 후 -1.5~-3% 되돌림, 구름대 위만, 양봉진입 금지). 잡주(소형·세력주) = 돌파 양봉 / 음봉 아래꼬리, 시가대비 -3%(우상향) ~ -5%(낙폭과대), VI 발동 후 시가 -3%.
- 백테스트: 부분(시총·되돌림%·224위는 코딩 가능; "꼬리"·구름대 추가 필요).
- 매핑: 신규 `candle_type_entry`(시총 분기 + 캔들색 필터) 또는 supertrend 레이어.
- 인용: wKWlWFtYYv8 "시총 1조 넘는 우량주는 무조건 음봉" / b-SZM-atWgU "시가 -3% 밀리면 매수".

### 그룹 F — 청산 / 회피 신호

**JD-R13 · 거래량 300% 장대음봉 distribution(이탈) 경보** — *신규, 즉시 코딩 가능*
- 정의: 정배열 확장구간(상승추세 중) 전일比 거래량 **≥300%** 장대음봉 = 세력 분산(distribution) → 당분간 하락. 보유 청산 신호.
- 백테스트: **가능** (OHLCV만으로 완결; 음봉 몸통 임계만 보정 필요).
- 매핑: 신규 `distribution_alert`(거래량 배수 + 음봉 + 정배열) → SELL/회피. (현 시스템에 distribution 청산신호 전무.)
- 인용: -PLoE2xLUPU "전일比 300%+ 장대음봉 출현=당분간 하락".

**JD-R14 · 기준봉(쿠데타봉) 시가 손절** — *가능*
- 정의: 하락추세 중 첫 대형 양봉=기준봉. 시가 이탈=손절 / 유지=추세전환 분할매수.
- 백테스트: 가능. 인용: tQ0eW3Qj74g "기준봉 4,200원 지켜주면 됨, 이탈시 손절".

### 그룹 G — 눌림목 (대부분 더트레이딩과 중복)

**JD-R15 · 다중지지 confluence 눌림** — *중복+α*
- 정의: 상승분 Fib 0.5/0.618 되돌림 + 5일선 + 전일고점 + 박스상단 중 **2개 이상 겹침** 시 매수(겹칠수록 강도↑). 손절 ~1~3%.
- 매핑: `f_zone`/`gold_zone`(이미 Fib 기반)에 **confluence count 가중 레이어**가 신규 가치.
- 인용: sRvURsAozgs "0.5·0.618…기준선·박스상단 세 근거 겹침".

**JD-R16 · 박스돌파 리테스트** — *중복+α*
- 정의: 박스상단 돌파+안착 → 저항이 지지로 전환된 자리 눌림 재진입. 1차 목표=박스높이 70%, 2차=박스 고점. 손절=장대양봉 저가/박스상단 이탈.
- 매핑: sf_zone/f_zone의 retest 변형(목표가 70% 공식이 신규).

### 그룹 H — 수급 / 주도주 / 국면 (2026 데일리)

**JD-R17 · 주도주 쇄빙선→순환매** — *부분*
- 정의: 외인이 대형 반도체주(삼전·하닉)에 선집중="쇄빙선" → 신고가 돌파 후 후방 섹터 **순환매**. 넥스레소 시초 +10% = 당일 대장주.
- 매핑: `leader-stock-scorer`(상위 2~5종목 52주 고가 돌파 → 후방 섹터 스코어업). 비공개 "수박지표" 의존부는 제외.
- 인용: BfQ9Z_s7fy0 "삼성전자·하이닉스는 쇄빙선…뒤 종목 순환매".

**JD-R18 · 수급게이트 (신용잔고·ETF·쏠림)** — *신규, 가능*
- 정의: ① 코스피 상위종목 거래대금 쏠림 → 코스닥 신규진입 억제 ② 신용잔고비율 고점 → 매수 억제 ③ 테마 ETF 상장 = 단기 고점 경보.
- 백테스트: **가능** (KRX 공개데이터: 거래대금 점유율·신용잔고비율·ETF 상장 이벤트). 임계는 미명시 → 캘리브레이션 필요.
- 매핑: 신규 `supply_credit_gate`. 인용: qWy7kdCOwaw "ETF 나오면 고점" / JbI_3CsQ1PM "신용비율 높을 때 주가 안 감".

**JD-R19 · 장세 4단계 레짐** — *부분(macro_regime 확장)*
- 정의: 주도주 → 실적주 → 테마주 → 역실적 장세 순환. 공격/방어 전환 조건(코스피 5일선·신용비율·주도주 5일선 이탈).
- 매핑: `market_regime.py` 확장(4단계 분류). 전환 임계 미명시 → 추가 피처 필요.

**JD-R20 · 오돌리 (5MA>15MA 크로스) 타점** — *가능, blue_line 유사*
- 정의: 5일선이 15일선을 위로 돌리는 시점 단타·스윙 진입. 224 위에서만.
- 매핑: `blue_line`(현 5/20 cross) 파라미터 변형(5/15) + 224 게이트.

### 그룹 I — 자금관리 / 운영 노하우 (정량 규칙 JD-R21~25 + 메타원칙 JD-P)

**JD-R21 · 최소 손익비 R:R ≥ 1:2 (권장 1:3) 게이트** — *가능, 최고 ROI*
- 정량: 진입 전 손절/목표 설정 후 비율 확인, 미달이면 NO-GO. 손절 0.6~1%→목표 ≥2~3%, 손절 1.8%→목표 ≥3.6%.
- 매핑: `risk_manager.min_rr_ratio = 2.0` (ATR 손절폭 대비 목표폭). 인용: Pc5P3RTntF0 "적어도 1대3".

**JD-R22 · 분할매수 비율** — *가능*
- 정량: 단타 5:5 / 공격형 2:3:5 / 보수형 4:6·3:7·1:1:1. 중장기일수록 분할수↑.
- 매핑: `position_sizing.split_ratio_*` (종목성격 분기). 인용: AIdNBufBk20 "2대3대5", Gi-Yy186pxg "최소 4대6".

**JD-R23 · 칼손절(이탈 즉시) + 사전 시나리오 필수** — *가능 + 메타*
- 정량: 손절 이탈 즉시 체결("2% 아까워 버티다 17%"). 진입 전 손절가·익절가·목표기간 3종 필수.
- 매핑: `exit.stop_loss_trigger="intraday_close"`; 주문 생성 시 stop/target/max_hold null이면 거부.

**JD-R24 · 몰빵 금지** — *가능*
- 정량: 단일 종목 1회 주문 ≤ 가용자금 50%. 매핑: `position_sizing` 1회 한도. 인용: Gi-Yy186pxg "몰빵하면 -30~40%".

**JD-R25 · 고점추격 진입 금지** — *가능*
- 정량: 장대양봉 직후 즉시 시장가 금지(눌림 대기); 갭 ≥10% 당일추격 자제; 바닥대비 +30% 신규진입 금지; 거래량 3배 + RSI>70 진입금지로 수치화.
- 매핑: `risk_manager` 진입금지 필터. (※ 더트레이딩 R11 "진입가 위치 게이트"와 동일 취지 — 교차검증.)

**농사매매 (분할 마틴게일)** — ⚠️ **권고 제외**
- 정의: 20~30종목 분산, 1차 1% → 1/2/4/8 배수 물타기(직전 대비 -15~20% + 종가양봉), 신저가 손절, 현금 ≥30%, 6개월 한도.
- 백테스트는 가능하나 **마틴게일 계열**(본인도 "자본 무한대 가정") → 폭락 장기화 시 자본고갈. **BarroAiTrade 거버넌스상 도입 비권고**(고위험). 신저가 손절·섹터분산·현금비중만 *원칙*으로 참고.

**메타원칙 JD-P** (정성, 운영 반영):
- JD-P1 매수 전 시나리오 확정(손절/익절/기간 3종) → 주문 필수필드.
- JD-P2 손절 불이행=구조 문제 → 스탑로스 자동화(시스템 강점).
- JD-P3 일봉 자리 먼저, 분봉 타이밍 나중 → 일봉 게이트 통과 후 분봉 스캔.
- JD-P4 몰빵 금지 → 분할 강제(=JD-R24).
- JD-P5 "갈 놈만 지지" → 지지 구간서 거래량/체결강도 급감 시 자동 손절.
- JD-P6 우하향 종목 중기 전환 금지 → 하락추세 전환 시 max_hold 강제 단축.
- JD-P7 기준 일관성(노이즈 무시) → PDCA로 사전기준 vs 실제 괴리 기록.

## 5. 비공개·재량 요소 (전이 불가, 명시)

- **수박지표 / 파란 점선**: 자체 HTS 전용 인디케이터 → 공개 발화로 복원 불가. (볼린저 수축→팽창으로 대리 가능하나 동일성 보장 X.)
- **검색기 "가린 버전"**: JD-R10/R11 일부 세부 항목 화면 가림 → 공개 항목만 사용.
- **밥그릇 1/2/3/4번 자리 구분**: 후행 레이블 → state-machine 근사 필요(실시간 정밀도 한계).
- **체결강도 110·호가잔량 150%**: 키움 전용 계산/L2 실시간 → 라이브 전용(백테스트 제약).
