# 슈퍼트렌드(Supertrend) 매매전략 상세 분석 리포트

- 작성일: 2026-06-01 · 분석: 운영 전략코드(`compute_supertrend`/`compute_adx`) 직접 호출 결정적 백테스트 + 적대적 자기검증
- 데이터: 5분봉 캐시 — **거래대금 상위 50종목**, **2026-04-23 ~ 2026-05-29** (종목당 약 1,800봉 ≈ 24거래일)
- 비용: 수수료 0.015%(매수·매도 각) + 거래세 0.18%(매도) **차감 순수익** (매수원가 = price×1.00015)
- 체결: 신호 확정 봉의 **다음 봉 시가** 진입/청산 (look-ahead 없음)
- 코드 기준: main `1f4c8de` (FLIP 휩쏘 필터 정정 반영분)

> ⚠️ 본 전략은 **signal-only**(실주문 미연결)이며, **슈퍼트렌드 전용 검증 모드**(타 전략 sf/f/gold 일시 비활성)로 운용 중입니다.

## 1. 전략 개요

TradingView Pine Script *Supertrend* 결정적 이식. **5분봉** 추세전환 추종(롱 온리).

| 항목 | 값 |
|---|---|
| 중심선(src) | hl2 = (고가+저가)/2 |
| ATR | RMA(Wilder) ATR(10) |
| 밴드 배수 | ×3.0 |
| 상단밴드 dn | src + 3·ATR (저항/하락추세선) |
| 하단밴드 up | src − 3·ATR (지지/상승추세선) |
| 유니버스 | 당일 거래대금·거래량 순위 (`RankUniverseProvider`) |

**밴드 캐리오버(lock)**: 직전 종가가 직전 상단밴드 위면 `up=max(up,up₁)`, 직전 종가가 직전 하단밴드 아래면 `dn=min(dn,dn₁)` — 추세선이 한 방향으로만 조여지는 트레일링 구조.

## 2. 신호 정의

- **BUY**: `trend −1→+1` 전환 봉 (종가가 직전 상단밴드 dn₁ 상향 돌파)
- **SELL**: `trend +1→−1` 전환 봉 (종가가 직전 하단밴드 up₁ 하향 이탈)

진입/청산 **완전 대칭**. 사용자 요청 반영으로 **장마감 강제청산 제거**(`time_exit=None`) — SELL 발생 시에만 매도.

## 3. 진입/청산/필터 게이트

| 게이트 | 기본값 | 설명 |
|---|---|---|
| entry_lookback | 2봉 | 최근 2봉 내 BUY + 현재 상승추세일 때만 진입 |
| exit_lookback | 2봉 | 최근 2봉 내 SELL일 때만 청산 |
| min_adx | 0(off) | ADX(14)<임계면 진입거부 — 횡보 차단 |
| min_flip_atr_mult | 0(off) | 전환 봉이 **방금 돌파한 저항(직전 dn밴드)** 을 mult·ATR 이상 넘어야 진입 (정정됨) |
| min_atr_pct | 0(off) | ATR%% 미달(저변동) 진입거부 |
| entry_time_cutoff | None | 장 후반 신규진입 차단(운영 override) |

## 4. 핵심 성과 — BASE (필터 OFF)

> 추세추종은 누적/복리 수치가 가정에 민감 → **거래당 통계(PF·승률·손익비)를 1차 지표**로.

| 지표 | 값 |
|---|---|
| 총 거래수 | 1042 (거래발생 50/50종목) |
| 승률 | **35.4%** |
| 거래당 평균 순수익 | **+0.993%** (중앙값 -0.850%) |
| **Profit Factor** | **1.762** |
| 평균 이익 / 평균 손실 | +6.48% / -2.02% |
| 손익비 | **3.21 : 1** |
| 최고 / 최악 단일거래 | +81.2% / -12.3% |
| 평균 보유 | 41.8봉 ≈ 3.5h (중앙 2.6h, 최장 19h) |
| 미청산(open_eod) | 29건 |

**해석.** 승률 35%로 낮지만 손익비 3.2:1 — 이익을 길게 끌고 손실을 짧게 끊는 전형적 추세추종. **PF 1.76은 비용 차감 후에도 거래당 기대값이 뚜렷한 (+)**.

**포트폴리오 관점.** 종목별 자기복리(같은 종목 1포지션 순차) **중앙값 +14.0%**, 평균 +23%(소수 강추세 종목이 왜곡), **양(+) 종목 35/50**. 평균이 아니라 **중앙값**을 대표값으로. 24거래일 강세장 구간 + 풀시드 가정이라 실전 기대치는 더 보수적으로 봐야 함.

## 5. 휩쏘 필터 비교 (FLIP 정정 후)

| 구성 | 거래수 | 승률 | 거래당평균 | PF | 평균이익 | 평균손실 | 미청산 |
|---|---|---|---|---|---|---|---|
| BASE(필터OFF) | 1042 | 35.4% | +0.993% | 1.762 | +6.48% | -2.02% | 29 |
| FLIP≥0.5 | 472 | 37.5% | +0.858% | 1.650 | +5.81% | -2.11% | 9 |
| FLIP≥1.0 | 245 | 43.7% | +1.212% | 1.997 | +5.56% | -2.16% | 2 |
| ADX≥25 | 397 | 39.5% | +1.085% | 1.828 | +6.06% | -2.17% | 14 |
| ADX≥25+FLIP≥0.5 | 184 | 42.4% | +1.040% | 1.783 | +5.58% | -2.30% | 6 |

**✅ FLIP 필터 정정 완료(main `1f4c8de`).** 종전엔 `breakout = close − 현재 up밴드`로 측정 → BUY 전환 직후 up밴드가 방금 `src−3·ATR`로 생겨 이탈폭이 항상 ~3·ATR → 0.5·ATR 문턱은 99.5% 무조건 통과(=필터 OFF와 동일, 무력)였습니다. **방금 돌파한 저항(전환 직전 dn밴드) 대비** `close − dn₁`로 정정하여 게이트가 실제로 작동합니다:

- FLIP≥0.5: 거래 1042→472건, 승률 37.5%
- **FLIP≥1.0: 거래 245건, 승률 43.7%, PF 2.00** (강한 전환만)
- ADX≥25: 거래 397건, 승률 39.5%, PF 1.83

**운영 권고**: 1차 적용은 **ADX≥25** 또는 **FLIP≥1.0**(보수적이면 둘 다). 필터를 켜면 거래수가 줄며(횡보 제거) 거래 품질(승률·PF)이 개선됩니다. 임계가 과하면 추세 초입 진입을 놓치므로 거래수 감소폭을 함께 모니터링.

## 7. 강점 / 약점

**강점**
- 추세 포착 시 손익비 3:1 — 큰 추세 1회가 다수 소손실 상쇄
- 진입/청산 동일 지표·동일 봉 대칭 → 규칙 단순·검증 용이, 룩어헤드 없음
- Pine 원본 충실 이식 + 전 구간 결정적 → 운영-백테스트 일치

**약점**
- 낮은 승률(35%) — 연속 손절 구간 드로다운·심리 부담
- 청산 후행 — SELL은 추세선 이탈 후 발생, 급락 시 추세선까지 손실 전부 수취
- `time_exit=None` → SELL 미발생 시 무기한 보유(오버나잇/갭 리스크). open_eod 29건이 사례

## 8. 리스크 점검

- **오버나잇 노출**: 장마감 강제청산 없음 → SELL 미발생 시 익일 이월(갭 리스크). 리스크엔진 강제청산/회로차단기 병행 검토.
- **단일 종목 집중**: 순위 유니버스 테마 쏠림 시 분산 저하 → balance_gate 균등분배 병행.
- **체결 슬리피지**: 백테스트는 다음봉 시가 가정. 실전은 폴링 종가 신호 → 지연·슬리피지 추가.

## 9. 코드 리뷰 / 변경 이력

1. **🟢 FLIP 휩쏘 필터 정정 (`supertrend.py`, main `1f4c8de`)** — `breakout` 기준을 현재 up밴드 → 직전 dn밴드(돌파한 저항)로 변경. 테스트 40건 통과. (본 리포트가 정정 효과를 반영)

3. **🟢 entry/exit_lookback=2** — 중복주문은 스캐너 구조로 차단(미보유만 진입/보유분만 청산).
4. **🟢 signal-only** — 진입·청산 모두 알림까지, 실주문 미연결. 현재 운영 안전.
