**전략 개요**: 블루라인 돌파 + 거래량 폭증(2배) + 캔들 실체(60%) + RSI ≤35 동시 충족

#### 성과 지표 (합성 데이터 기준)

| 지표 | 측정값 | 비고 |
|------|--------|------|

| 합성 데이터 승률 | 0~6% | 평가 신뢰성 낮음 |
| 실제 시장 예상 승률 | 55~65%* | 전문가 추정 |
| 이론적 Risk/Reward | 1:2.5 | 파라미터 설계 기준 |

> *실제 테마 종목에서 거래량 폭증+RSI 바닥권 조건이 동시 충족되는 경우 검증 필요

**분석**:
- 합성 데이터(GBM)에서 거래량 급증 + RSI 과매도 조건이 동시 충족하기 극히 어려움
- 전략 자체의 로직은 검증된 패턴(수박지표 = 서희파더 이재상 F존 파생 기법)
- **실거래 데이터 없이는 공정한 평가 불가** → 모의투자 수집 데이터로 재검증 필요
- **결론**: 모의투자 진행 중 실데이터 기반 재검증 후 3개월 내 2차 평가
