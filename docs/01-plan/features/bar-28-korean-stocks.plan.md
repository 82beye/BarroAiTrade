# BAR-28 한국 주식 시장 분석 + 관심종목 리스트 Plan

> **Project**: BarroAiTrade

> **Status**: In Progress
> **Gate**: Simulation Mode Launch (모의투자 개시)

---

## 1. 목표 및 범위

### 1.1 개요
한국 주식 시장의 주요 지표를 분석하고, BarroAiTrade 시스템에 최적화된 관심종목 리스트를 구성합니다. 이는 모의투자 개시 전 시장 적합성 검증을 위한 **Critical Gate**입니다.

### 1.2 주요 목표 (MoSCoW)

#### Must Have (필수)
- [ ] 한국 주식 시장 현황 분석 (KOSPI/KOSDAQ 지표)
- [ ] 시스템 거래 성능에 최적화된 종목 필터링 기준 수립
- [ ] 추천 관심종목 리스트 (30-50종목) 생성
- [ ] 종목별 거래 특성 분석 (유동성, 변동성, 거래량)

#### Should Have (권장)
- [ ] 산업군별 분류 및 포트폴리오 구성 제안
- [ ] 시즈널 트렌드 분석
- [ ] 기술적 강점(파란점선/수박신호) 맞춤 분석

#### Could Have (선택)
- [ ] 실시간 모니터링 대시보드 설계
- [ ] 관심종목 추가 기준 정의

### 1.3 성공 기준 (Definition of Done)
- ✅ 시장 분석 리포트: 최소 5개 섹션
- ✅ 관심종목 리스트: 30-50종목 + 이유 문서화
- ✅ 검증 완료: BAR-33(E2E) 통과 준비
- ✅ PDCA 완료율: 90% 이상

---

## 2. 범위 상세 분석

### 2.1 분석 대상 시장

| 시장 | 범위 | 주요 지표 |
|------|------|---------|
| **KOSPI** | 유가증권시장 | 지수, 거래량, 변동성 |
| **KOSDAQ** | 코스닥 | 소형주, 성장주 |
| **테마/ETF** | 섹터 분석 | 산업군별 강도 |

### 2.2 분석 지표

#### 시장 건강도
- KOSPI/KOSDAQ 지수 추세
- 상승/하락주 비율
- 거래량 추이
- VIX 지수 수준

#### 종목 선정 기준
- 유동성: 일평균 거래량 > 1M 주
- 변동성: 월간 변동률 5~30% 범위
- 기술적: 파란점선/수박신호 호응도
- 거래가능성: 예탁금 상한 내 거래 가능

#### 포트폴리오 구성
- 산업군 다각화 (5~8개 섹터)
- 시가총액 혼합 (대형/중형/소형)
- 위험/수익 균형

## 4. 데이터 소스 및 방법론

### 4.1 데이터 소스
- **KiwoomAPI**: 실시간 주가, 거래량
- **통계**: 월 평균, 변동률 계산
- **시스템 로그**: 파란점선/수박신호 히스토리
- **공개정보**: 산업분류, 시가총액

### 4.2 분석 방법론

#### 1단계: 시장 현황 분석 (2-3일)
```
KOSPI/KOSDAQ 추이 → 섹터별 강도 → 유동성 평가
```

#### 2단계: 종목 필터링 (3-5일)
```
조건부 스크리닝 → 기술신호 점수 → 거래특성 검증
```

#### 3단계: 포트폴리오 구성 (2-3일)
```
산업군 배분 → 리스크 조정 → 최종 리스트 확정
```

#### 4단계: 상세 분석 (3-5일)
```
개별 종목 분석 → 진입전략 제시 → 문서화
```
