### 진단

5/22 LG전자 손실의 원인은 **저변동이 아님**:
- 일봉 변동성 충분 (10.20%)
- 운영 시그널(SignalScanner) 진입은 5m 분봉 기준 — 일봉 ATR% 와 별개일 수
  있으나, 일봉이 이 정도면 분봉 ATR% 도 차단 임계 미만일 가능성 낮음
- 즉 변동성 필터 의도대로 작동

**진짜 손실 원인 후보**:
1. **진입 시점 문제** — 장 후반(13:48, 14:58) 진입 → 청산 여유 시간 부족
2. **청산 정책 문제** — TP=-100%, 트레일링 3.0%/-1.5% 설정상 일중 익절 기회
   놓침 (장 마감 시 강제 청산)
3. **종목 자체 추세** — LG전자가 그날 −2.60% 마감하는 약세 흐름

→ **Phase 8+ 의 청산 정교화 (PRD §9 원래 Phase 6) 가 우선순위 1순위로 부각**.
