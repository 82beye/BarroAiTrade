# 공구리·distribution OOS 검증 리포트

> **연관**: [[2026-06-21-dante-uplift.report|(c)구현+shadow 백테스트]] · [[../../02-design/features/2026-06-21-dante-uplift.design|접목 설계]]
>
> **Summary**: shadow 백테스트가 불장(2024~26) 편향이라는 한계를 받아, `sr_flip`(공구리)·`distribution`을 **3축 OOS**(기간 분할·종목 분할·파라미터 민감도)로 검증. 형식 게이트는 **둘 다 PASS**지만 분기·baseline 대비 분해 결과: **distribution은 시장베타가 아닌 견고한 상대-약세 엣지**, **공구리는 순익 대부분이 롱-베타로 baseline 대비 알파가 불안정한 조건부 PASS**(고유가치=약세분기 낙폭 축소). **진정한 약세장 OOS는 데이터셋 한계로 불가** → (d) 전 약세장 dry-run 필수.
>
> **Date**: 2026-06-22 · **Status**: Done(검증). (d) 라이브 활성은 HITL(§5).

## 1. 방법

- 데이터: `data/ohlcv_cache` 일봉, 2,670종목(≥460봉), 2023-11~2026-06. 224 EMA warmup → **평가구간 ≈2024-10~2026-06**.
- 스크립트: `scripts/backtest_dante_oos_validation.py`(관측 전용, 라이브 무영향).
- 3축 + `_oos_validation.py` 정신(trades≥30·holdout>0·부호안정):
  1. **기간 분할** — IS(date<2025-09-01) vs OOS(≥) + 분기별 breakdown.
  2. **종목 분할** — `Σord(sym)%2` disjoint 두 세트 양쪽 유지.
  3. **파라미터 민감도** — sr_flip lookback∈{15,20,25}, distribution vol_mult∈{2.5,3,3.5}·body_min∈{2,3,4%}.
- 진입(sr_flip): net=forward H20 − 왕복 0.90%. 회피(distribution): forward H10 이 baseline보다 낮고 음수면 유효.
- **핵심 한계**: warmup이 첫 ~1년 소진 + 잔여구간이 불장 → **약세장 OOS 불가**. baseline 대비 Δ 와 약세분기(2026Q2) 거동으로 *간접* 판단.

## 5. (d) 라이브 활성 권고 (HITL — 자동화 금지)

1. **distribution 청산/회피 (JD-R13)** — 가장 견고. `holding_evaluator`/exit 보조 청산 신호로 (d) 후보 1순위. 임계는 vol_mult≥3.0·body_min≥3%(단조성상 보수적일수록 강함). `barrotrade-code-surgeon` 위임 + config-gated default-OFF.
2. **공구리 sr_flip 진입 (JD-R7)** — 단독 활성 **비권고**. 순익이 롱-베타라 단순 롱 대비 알파 불명확. **레짐/추세 필터 결합**(예: `above_ma224`·정배열 게이트로 약세분기 진입 억제) 후 재측정 → dry-run 1~2주 → (d). 약세분기 낙폭 축소 특성을 살리려면 **하락장 비중 축소 룰**과 결합.
3. **공통 전제**: 본 OOS는 불장 구간 한정. **약세장 포함 장기데이터 또는 라이브 dry-run 통과 전 실거래 자본 편입 금지**(★_oos_validation 원칙).
