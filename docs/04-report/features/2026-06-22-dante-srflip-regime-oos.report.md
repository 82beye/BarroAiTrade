# sr_flip(공구리) × 레짐 필터 결합 OOS 재검증

> **연관**: [[2026-06-22-dante-oos-validation.report|공구리·distribution OOS]] · [[2026-06-21-dante-uplift.report|(c)구현]] · [[../../02-design/features/2026-06-21-dante-uplift.design|설계]]
>
> **Summary**: 1차 OOS에서 sr_flip 이 "롱-베타·알파 불안정"으로 조건부 PASS였던 점을 받아, **레짐 필터(above_ma224·정배열·above_ma60)** 결합 후 재검증. 결과: 레짐 필터는 **절대 net 상승 + 약세분기 방어를 크게 개선**하나, **regime-matched baseline 대비 순수 알파(regimeΔ)는 여전히 ≈0·부호 불안정** → 3변형 모두 형식 게이트 FAIL. **sr_flip은 레짐 결합 후에도 "상승추세 종목 베타 수확"이지 패턴 고유 알파가 아니다.** 권고: sr_flip은 (d) 진입 전략으로 승격하지 않고 관측/inert 유지. **distribution이 dante 시그니처 중 유일한 (d) 적격 엣지로 확정.**
>
> **Date**: 2026-06-22 · **Status**: Done(재검증). sr_flip (d) 진입 = 비권고.

## 1. 방법
- `scripts/backtest_dante_srflip_regime_oos.py`(관측 전용). 일봉 2,659종목(≥480봉), H20, 비용 0.90% 차감.
- 필터 F: none(base) / above_ma224(JD-R1) / 정배열(EMA 112>224>448) / above_ma60(완화).
- 비교 2기준: ① 무조건 baseline ② **regime-matched baseline**(같은 F 만족 봉의 forward) ← 알파 분리.
- 축: IS/OOS(2025-09-01) + 분기 + 종목 분할 + lookback 민감도. 판정: net>0 ∧ **regimeΔ>0**(IS·OOS) ∧ 종목 양쪽>0.

## 2. 결과 (net H20, 비용 차감)

| 변형 | N | net | uncond Δ | regimeΔ 전체 | regimeΔ IS/OOS | 2026Q2 net (vs regime-base) |
|------|---|-----|----------|-----------|---------------|------------------------------|
| 공구리(필터없음) | 23,792 | +1.784% | −0.004% | −0.004% | −0.446% / +0.491% | −2.068% (Δ+2.181) |
| **+above_ma224** | 18,477 | +1.954% | **+0.167%** | +0.047% | −0.145% / +0.244% | **−0.979%** (Δ+1.723) |
| +정배열 | 5,342* | +2.889% | +1.102% | +0.072% | (IS N=0) / +0.072% | **+1.218%** (Δ+2.597) |
| +above_ma60 | 23,033 | +1.820% | +0.033% | +0.187% | −0.008% / +0.385% | −1.875% (Δ+2.165) |

\*정배열은 448 EMA warmup으로 평가구간이 늦어져 **IS 표본 0**(전부 OOS) — 시간 OOS 검증 불가.

- 파라미터 민감도(+above_ma224): lookback 15/20/25 = +1.962/+1.954/+2.024% (견고).
- 종목 분할: 전 변형 두 세트 net 양수(세트0 < 세트1 일관, 종목쏠림 아님).
- 분기별(+above_ma224 vs regime-base): 2024Q4 −1.45 · 2025Q1 −1.40 · 2025Q2 +0.23 · 2025Q3 −0.49 · 2025Q4 +0.08 · 2026Q1 −0.48 · **2026Q2 +1.72** → regimeΔ 부호 분기마다 진동.

## 3. 해석

1. **레짐 필터는 절대 성과·드로다운을 개선한다.** above_ma224는 net +1.784→+1.954%(uncond Δ+0.167%), 정배열은 +2.889%(Δ+1.102%). 약세분기(2026Q2)는 base −2.07% → above224 −0.98%, **정배열은 +1.22%로 양전환**. 상승추세 종목으로 진입을 한정해 *추락 종목 진입을 회피*한 효과.
2. **그러나 순수 알파는 없다.** regime-matched baseline 대비 regimeΔ는 above224 +0.047%·정배열 +0.072%·above60 +0.187%로 **≈0**, 게다가 IS/OOS·분기 부호 불안정. 즉 개선분의 대부분은 "상승추세 종목이 더 오른다"는 **베타/유니버스 선택 효과**이지 공구리 패턴 자체의 종목선택 알파가 아니다. → 같은 효과를 *단순히 above_ma224 롱*으로도 얻는다.
3. **형식 게이트 FAIL(3변형 모두).** regimeΔ IS<0(또는 정배열 IS 표본 0)으로 "레짐 내 안정적 양의 알파" 조건 미충족.

## 4. 결론·권고

- **sr_flip은 (d) 진입 전략으로 승격하지 않는다(비권고).** 레짐 결합 후에도 regime-matched baseline을 못 이긴다(순수 알파 ≈0). 라이브 진입 신호로서 가치 불충분.
- sr_flip은 **관측/inert 유지**(dante_filters의 순수함수로 보존, 호출처 없음). 향후 *드로다운 축소 보조 신호*나 연구용으로만.
- **distribution(JD-R13)이 dante 시그니처 중 유일한 (d) 적격 엣지로 확정** — 1차 OOS에서 regime-matched 무관하게 baseline 하회·음수·임계 단조(진짜 상대-약세). 이미 (d) config-gated default-OFF 구현·머지(PR #189).
- (참고) 정배열 2026Q2 양전환(+1.22%)은 흥미롭지만 **OOS-only·소표본**이라 결론 아님. 약세장 데이터 확보 시 재검토 후보.
