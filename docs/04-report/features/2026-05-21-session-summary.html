    ✅ SHORT_TERM_HIGH (단기 고점 도지·위꼬리)
    ❌ breakeven / STOP_LOSS (-4~-5%) 차단

  P1 partial_tp_done 보존 (분할매도 폭주 방지)
  P2 trough_pnl_rate 기반 DCA
  P3 매수만 한도 카운트 (매도 항상 허용)
